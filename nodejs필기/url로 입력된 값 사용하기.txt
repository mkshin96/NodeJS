http(protocol 통신규칙) : 사용자가 서버에 접속할 때 어떤 방식으로 통신할 것인가에 대한,
                         웹브라우저와 웹서버가 데이터를 주고받기 위해 만든 통신규칙

host(domain) : 어떤 특정한 인터넷에 연결되어 있는 컴퓨터를 가르키는 주소

port

path : 컴퓨터 안의 어떤 디렉토리의 어떤 파일인지 가르킴

query string : 값을 변경하면, 웹 서버에게 어떤 데이터를 전달할 수 있다.
               시작은 ?로 하기로 약속, 값과 값은 &, 이름은 = 로 구분.
